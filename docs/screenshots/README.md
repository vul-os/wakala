Both images are committed so the README renders on GitHub without requiring a
local screenshot run.

## Regenerate

```bash
# From repo root:
npm run screenshots

# Or directly:
cd scripts && npm ci && node screenshots.mjs
```

Prerequisites: Node.js 20+. Playwright downloads a headless Chromium binary
(~170 MB) on first `npm ci` in `scripts/`.

## Notes

- The demo harness (`demo/index.html`) is self-contained — no real backend
  required. It uses in-memory stub peers via `BroadcastChannel` simulation.
- The SVG sequence diagram is embedded directly in the HTML and does not depend
  on any external rendering service.
- If Chromium cannot be launched (CI with no display, sandbox restrictions),
  the script will fail gracefully with a descriptive error. In that case,
  run locally and commit the outputs.
